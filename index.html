This is a hierarchical navigation system that demonstrates multi-level menu structures:

Navigation Flow:

1. Click on "Projects" to see available projects

2. Click on "PPO Project" to see the module options

3. Click on any letter (A-H) to navigate to that specific module

Each letter button would typically navigate to a separate HTML file like ppo-module-a.html, ppo-module-b.html, etc.
