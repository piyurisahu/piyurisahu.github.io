MICROAPP DEPLOYMENT ARCHITECTURE
Micro Front-end
The Micro Front-end is a JavaScript file served up using an HTTP server.
Outer API
The Outer API is based on the Micron framework and is tightly coupled to the Micro Front-end that renders the UX that this Outer API delivers data and functionality to. Calls to the Outer API come from the user device. For example, in Web Based Banking, it will be the Web Browser which runs the Micro Front-end which makes calls to the Outer API.
Write Inner API
Read Inner API
The Inner API is based on the Micron framework and is coupled to a System of Record (SOR). We leverage the CQRS design pattern so there will be 2 Inner APIs per SOR we want to access. One Inner for Reads and one for Writes. The Inner for Reads will go to Bob while the Inner for Write will transact with the SOR.
